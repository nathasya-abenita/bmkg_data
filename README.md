# BMKG Meteo Database

This is created to process [daily meteorological data](https://dataonline.bmkg.go.id/dataonline-home) published publicly by BMKG (Indonesian Meteorology, Climatology, and Geophysical Agency). Note that the available public data is prepared in Excel file.
